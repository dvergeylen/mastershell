> Interfaccia di controllo del server HTTP Apache per macOS.

#### Avvia il demone `org.apache.httpd`:
```shell
apachectl start
```
#### Ferma il demone:
```shell
apachectl stop
```
#### Riavvia il demone:
```shell
apachectl restart
```
