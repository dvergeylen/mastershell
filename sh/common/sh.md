> Bourne ljuska.
> Standardni interpreter komandnog jezika.
> Više informacija: <https://manned.org/sh>.

#### Pokreni interaktivnu ljusku:
```shell
sh
```
#### Izvrši komandu:
```shell
sh -c {{komanda}}
```
#### Pokreni komande iz datoteke:
```shell
sh {{datoteka.sh}}
```
#### Pokreni komande iz stdin-a:
```shell
sh -s
```
