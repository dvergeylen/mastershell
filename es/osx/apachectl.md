> Interfaz de control pata Apache HTTP Server para macOS.

#### Inicia la tarea launchd `org.apache.httpd`:
```shell
apachectl start
```
#### Finaliza la tarea launchd:
```shell
apachectl stop
```
#### Finaliza e inicia la tarea launchd:
```shell
apachectl restart
```
