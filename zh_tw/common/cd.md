> 改變工作目錄。
> 更多資訊： <https://man.archlinux.org/man/cd.n>.

#### 前往指定目錄：
```shell
cd {{目錄/完整/路徑}}
```
#### 前往目前使用者的家目錄：
```shell
cd
```
#### 前往目前目錄的上層目錄：
```shell
cd ..
```
#### 前往之前所在的目錄：
```shell
cd -
```
