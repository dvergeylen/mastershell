> 退出终端程序。
> 更多信息见：<https://manned.org/exit>.

#### 使用最后执行命令的退出代码，退出终端程序：
```shell
exit
```
#### 使用指定的退出代码，退出终端程序：
```shell
exit {{exit_code}}
```
