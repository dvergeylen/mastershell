> 更改当前工作目录.
> 更多信息： <https://man.archlinux.org/man/cd.n>.

#### 转到指定目录:
```shell
cd {{路径}}
```
#### 转到当前用户的主（home）目录:
```shell
cd
```
#### 转到当前目录的父目录:
```shell
cd ..
```
#### 转到刚才选择的目录:
```shell
cd -
```
