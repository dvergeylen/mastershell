> 退出当前的命令行实例或 bat 脚本.

#### 退出当前的命令行实例:
```shell
exit
```
#### 退出当前的 bat 脚本:
```shell
exit /b
```
#### 使用一个指定的退出码退出:
```shell
exit {{退出码}}
```
