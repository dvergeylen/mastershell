> 用于在 Windows 注册表中管理键及其值的命令行界面.
> 参见 `reg-query`, `reg-add` 和其他的页面来获取额外的信息.

#### 执行注册表命令:
```shell
reg {{命令}}
```
#### 显示常规信息并列出所有可用命令:
```shell
reg /?
```
#### 显示指定命令的帮助信息:
```shell
reg {{命令}} /?
```
