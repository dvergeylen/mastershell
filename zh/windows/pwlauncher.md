> 用于管理 Windows To Go 启动选项的命令行工具.

#### 显示当前 Windows To Go 的状态:
```shell
pwlauncher
```
#### 启用或禁用 Windows To Go 的启动选项:
```shell
pwlauncher /{{enable|disable}}
```
