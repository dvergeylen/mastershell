> Chocolatey 包管理器的命令行界面.
> 参见 `choco install`, `choco upgrade` 和其他的页面来获取额外的信息.

#### 执行 Chocolatey 命令:
```shell
choco {{命令}}
```
#### 显示基本的帮助页面:
```shell
choco -?
```
#### 显示指定命令的帮助页面:
```shell
choco {{命令}} -?
```
#### 显示 Chocolatey 的版本:
```shell
choco --version
```
