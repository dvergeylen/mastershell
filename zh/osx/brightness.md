> 获取或设置所有显示设备的亮度等级.

#### 显示当前亮度:
```shell
brightness -l
```
#### 设置亮度到 100%::
```shell
brightness {{1}}
```
#### 设置亮度到 50%::
```shell
brightness {{0.5}}
```
