> 用于 macOS 的 Apache HTTP Server 控制接口（工具）.

#### 启动 org.apache.httpd 服务:
```shell
apachectl start
```
#### 停止已启动的服务:
```shell
apachectl stop
```
#### 重新启动服务:
```shell
apachectl restart
```
