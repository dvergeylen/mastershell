> Get and set the brightness level of all internal and certain external displays.

#### Show current brightness:
```shell
brightness -l
```
#### Set the brightness to 100%::
```shell
brightness {{1}}
```
#### Set the brightness to 50%::
```shell
brightness {{0.5}}
```
