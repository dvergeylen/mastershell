> Control macOS dark mode from the command-line.
> More information: <https://github.com/sindresorhus/dark-mode>.

#### Toggle dark mode (turn it on if it's currently off, off if it's currently on):
```shell
dark-mode
```
#### Turn dark mode on:
```shell
dark-mode on
```
#### Turn dark mode off:
```shell
dark-mode off
```
#### Check if dark mode is on:
```shell
dark-mode status
```
