> Manual page viewer for X Window System.

#### Start xman in three-button window:
```shell
xman
```
#### Open the manual page output stored in a given file:
```shell
xman -helpfile {{filename}}
```
#### Show both manual page and directory:
```shell
xman -bothshown
```
