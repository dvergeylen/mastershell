> A command-line tool for controlling NetworkManager.
> More information: <https://man.archlinux.org/man/nmcli.1>.

#### Check the nmcli version:
```shell
nmcli --version
```
#### Call general help:
```shell
nmcli --help
```
#### Call help on a command:
```shell
nmcli {{command}} --help
```
#### Execute an `nmcli` command:
```shell
nmcli {{command}}
```
