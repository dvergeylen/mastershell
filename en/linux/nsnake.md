> Snake game in the terminal.
> More information: <https://github.com/alexdantas/nsnake/>.

#### Start a snake game:
```shell
nsnake
```
#### Navigate the snake:
```shell
{{Up|Down|Left|Right}} arrow key
```
#### Pause/unpause the game:
```shell
p
```
#### Quit the game:
```shell
q
```
#### Show help during the game:
```shell
h
```
