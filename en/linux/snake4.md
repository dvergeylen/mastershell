> Snake game in the terminal.
> More information: <https://manpages.debian.org/snake4/snake4.6.en.html>.

#### Start a snake game:
```shell
snake4
```
#### Choose level:
```shell
{{1|2|3|4|5}}
```
#### Navigate the snake:
```shell
{{Up|Down|Left|Right}} arrow key
```
#### Pause game:
```shell
Spacebar
```
#### Quit game:
```shell
q
```
#### Show the high scores:
```shell
snake4 --highscores
```
