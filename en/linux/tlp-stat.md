> A tool to generate TLP status reports.
> See also `tlp`.
> More information: <https://linrunner.de/tlp/usage/tlp-stat>.

#### Generate status report with configuration and all active settings:
```shell
sudo tlp-stat
```
#### Show battery information:
```shell
sudo tlp-stat -b
```
#### Show configuration:
```shell
sudo tlp-stat -c
```
