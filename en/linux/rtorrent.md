> Download torrents over the command-line.

#### Add a torrent file or magnet to be downloaded:
```shell
rtorrent {{torrent_or_magnet}}
```
#### Start the download:
```shell
<Ctrl>S
```
#### View details about downloading torrent:
```shell
->
```
#### Close rtorrent safely:
```shell
<Ctrl>Q
```
