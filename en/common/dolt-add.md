> Add the contents of a table to the list of Dolt staged tables.
> More information: <https://github.com/dolthub/dolt>.

#### Add a table to the list of staged tables (stage a table):
```shell
dolt add {{table}}
```
#### Stage all tables:
```shell
dolt add --all
```
