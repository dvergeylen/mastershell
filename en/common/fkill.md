> Fabulously kill processes. Cross-platform.
> More information: <https://github.com/sindresorhus/fkill>.

#### Run without arguments to use the interactive interface:
```shell
fkill
```
#### Kill the process by pid, name or port:
```shell
fkill {{pid|name|:port}}
```
