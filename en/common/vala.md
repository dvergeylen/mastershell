> Vala code runner.
> Tutorial: <https://wiki.gnome.org/Projects/Vala/Tutorial>.
> More information: <https://valadoc.org/>.

#### Run a vala file, with gtk+:
```shell
vala {{path/to/file.vala}} --pkg {{gtk+-3.0}}
```
#### Display version info:
```shell
vala --version
```
#### Display helper message:
```shell
vala --help
```
