> Print the total number of commits.
> Part of `git-extras`.
> More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-count>.

#### Print the total number of commits:
```shell
git count
```
#### Print the number of commits per contributor and the total number of commits:
```shell
git count --all
```
