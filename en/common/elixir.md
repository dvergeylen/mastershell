> Elixir programming language interpreter.
> More information: <https://elixir-lang.org>.

#### Run an Elixir file:
```shell
elixir {{path/to/file}}
```
#### Evaluate Elixir code by passing it as an argument:
```shell
elixir -e "{{code}}"
```
