> Chocolateyパッケージマネージャーのコマンドラインインターフェイスです
> 詳細については、「choco install」、「choco upgrade」などのページを参照してください
> 詳しくはこちら: <https://chocolatey.org>.

#### Chocolateyコマンドを実行します:
```shell
choco {{コマンド}}
```
#### 一般的なヘルプを表示します:
```shell
choco -?
```
#### 特定のコマンドのヘルプを表示します:
```shell
choco {{コマンド}} -?
```
#### Chocolateyバージョンを表示します:
```shell
choco --version
```
